# Разработка интернет-приложений (РИП) 2025

## Тема: Расчет урожайности сада яблонь для указанных климатических условий в сезоне

Разработанное приложение позволит садоводам создавать и формировать заявки на расчёты урожайности сада яблонь, а агрономам завершать или отклонять данные заявки.
